Detailed Description

A point in time.

Represents a point in time as A + B/2^32 seconds where A is an int64 and B is a uint32.

Use seconds() to conver the fixed point representation to floating point seconds. Use now() to get the current HostPointTime. Internally now() uses std::chrono::steady_clock which is not related to wall time and increases monotonically (e.g. it might be the time since the last reboot).

Examples:
TimePointStreaming_CachedPoint.cpp, TimePointStreaming_Circle.cpp, TimePointStreaming_Focus.cpp, TimePointStreaming_Forcefield.cpp, TimePointStreaming_Freeform.cpp, TimePointStreaming_FreeformTracking.cpp, and TimePointStreaming_MultiPoint.cpp.
