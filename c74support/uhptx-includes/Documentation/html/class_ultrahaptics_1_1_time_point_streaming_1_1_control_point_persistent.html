Detailed Description

A control point that can be enabled or disabled and can belong to a control point group.

This class is the same as a ControlPoint but adds an "enabled" flag and control point group membership.

The control point can be disabled by calling disable(). In this state it will not be emitted.

You can set the control point group using setGroup(). See Introduction for more information about control point groups, and why they need to be specified here.
