Detailed Description

SI unit constants.

These provide a few convenience constants to convert from prefixed units and to allow annotating values with their unit.

These are very simple and do not use the type system to enfore correct use of units. Other libraries are available for this if desired, for example:

  • Boost's Unit library
  • UNITS
  • PhysUnits
