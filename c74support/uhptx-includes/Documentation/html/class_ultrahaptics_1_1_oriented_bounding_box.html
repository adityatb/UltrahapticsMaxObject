Detailed Description

OrientedBoundingBox is a parallelepiped in any orientation.

A parallelepiped could be specified by four vectors - a corner, and vectors for the three edges connected to that corner. However this class actually uses the centre instead of a corner, and the edge vectors are split into normalised vectors and lengths scalars.
